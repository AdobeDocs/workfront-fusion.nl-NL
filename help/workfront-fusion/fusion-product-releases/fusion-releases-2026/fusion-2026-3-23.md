## Een aangepaste API-aanroepmodule (Polling) nu beschikbaar in de Adobe Authenticator-connector

Om het mogelijk te maken om API vraag te herhalen tot een bepaalde voorwaarde of een grens wordt voldaan aan, hebben wij de Make een aangepaste API vraag (Opiniepeiling) module gecreeerd. Deze module bevat opties waarmee u API-aanroepen voor herhalingen kunt configureren. U kunt de API vraag hebben uitvoeren tot een bepaalde voorwaarde wordt voldaan, een specifiek aantal herpogingen plaatsen, en de tijd tussen herpogingen vormen.

Voor meer informatie, zie [ een douane API vraag (Opiniepeiling) ](/help/workfront-fusion/references/apps-and-modules/adobe-connectors/adobe-authenticator-modules.md#make-a-custom-api-call-polling) in de modules van artikel Adobe Authenticator maken.
